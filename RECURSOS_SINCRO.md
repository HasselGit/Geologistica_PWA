## 📌 Protocolo de Sincronización
Para cualquier IA nueva:
1.  Leer primero `sesion_actual.md` para el contexto inmediato.
2.  Consultar esta carpeta `sincronizacion/` para el conocimiento estructural.
3.  No reintroducir elementos gráficos redundantes y siempre ejecutar `flutter analyze` para verificar la estabilidad de compilación antes de finalizar.
